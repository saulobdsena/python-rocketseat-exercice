This is my first Python exercise from the **Rocketseat Python course**.

## About

The goal of this project is to build a simple **task manager (to-do list)** using core Python concepts such as dictionaries, functions, and loops.

## Features

- Add a new task
- List all tasks
- Mark a task as completed
- Remove a task
